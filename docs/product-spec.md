# 🧠 念头解钩器（Thought Unhook）— 产品详细方案

> **核心理念**：你不是你的念头。把脑内的声音"物化"出来，看见它、触碰它、放下它。

---

## 一、产品定位

### 1.1 一句话描述
一款帮助用户将脑海中纷飞的念头"外化"为可视化物体，通过物理化的交互操作（观察、标记、变形、释放）来实现与念头"脱钩"的心理健康工具。

### 1.2 目标用户
| 用户群体 | 痛点描述 |
|---------|---------|
| 焦虑反刍者 | 同一个念头反复循环，越想越焦虑 |
| 失眠人群 | 睡前脑内嘈杂，无法安静 |
| 高压工作者 | 思绪过载，难以聚焦当下 |
| 冥想初学者 | 知道要"观察念头"但不知怎么做 |
| 情绪敏感者 | 容易被一个想法带走，陷入情绪漩涡 |

### 1.3 核心价值主张
- **不是压制念头，而是"看见"念头** — 认知解离（Cognitive Defusion）
- **不是分析念头，而是"体验"念头消散** — 具身认知（Embodied Cognition）
- **不是心理治疗，而是日常心理卫生工具** — 像刷牙一样自然

## 二、核心概念：念头的"物化"模型

### 2.1 念头生命周期

```
[产生] → [捕捉] → [物化] → [观察] → [变形/消解] → [释放]
  ↑                                                    |
  └────────── 念头自然循环 ←────────────────────────────┘
```

### 2.2 念头的物理形态

每一个念头被输入后，会被渲染为一个**漂浮的半透明气泡/物体**，具有以下可感知属性：

| 属性 | 含义 | 视觉表现 |
|------|------|---------|
| **大小** | 这个念头占据你多少注意力 | 气泡尺寸 |
| **颜色** | 念头的情绪色彩 | 暖色=焦虑/愤怒，冷色=悲伤，灰色=无聊/麻木 |
| **密度** | 念头的"粘性"（越粘越难放下）| 透明度，越实心越粘 |
| **运动** | 念头的活跃程度 | 抖动频率、轨迹 |
| **文字** | 念头的内容 | 气泡内若隐若现的文字 |

### 2.3 认知解离的核心交互

用户与物化念头的交互方式，基于ACT（接纳承诺疗法）中的经典练习：

| 交互操作 | 心理机制 | 具体体验 |
|---------|---------|---------|
| 🫧 **看见** | 去融合 | 念头从脑内文字变为眼前漂浮物 |
| 🏷️ **贴标签** | 分类觉察 | "这是一个关于未来的担忧" |
| 🔍 **放大/缩小** | 透视转换 | 捏合手势让念头变大变小 |
| 🎵 **变声** | 去严肃化 | 用卡通音/慢速重新播放念头文字 |
| 💨 **吹走** | 释放 | 长按或吹气(麦克风)让气泡飘走 |
| 🫠 **融化** | 接纳消散 | 看着念头慢慢变透明直到消失 |
| 📌 **暂存** | 稍后处理 | 把念头放进"念头罐"，不急着处理 |

### 3.2 核心功能详述

---

#### 🫧 模块一：念头空间（Thought Space）— 主场景

**概述**：一个沉浸式的3D/2D空间，用户在这里"倾倒"脑内念头并与之互动。

**用户流程**：

```
1. 打开App → 进入「念头空间」
2. 看到一片安静的空间（星空/水面/虚空，可选主题）
3. 点击 ＋ 或语音按钮 → 输入一个念头
4. 念头以「气泡」形态出现在空间中
5. 可以继续输入更多念头（多个气泡漂浮）
6. 选择一个气泡 → 进入「解钩」交互
7. 通过各种操作与念头互动
8. 念头被释放/暂存 → 空间重归平静
```

**关键交互细节**：

**输入方式**：
- ✍️ 文字输入：简短句子描述念头（"我怕明天的会议搞砸"）
- 🎤 语音输入：直接说出来，AI转文字（更自然的"倾倒"感）
- 🤖 AI辅助捕捉：输入模糊感受，AI帮你精确命名（"说不清的难受" → "你可能在感受一种无力感？"）

**空间交互**：
- 气泡会缓缓漂浮、轻微碰撞
- 长按气泡 → 展开操作菜单（标签/变声/释放等）
- 双指捏合 → 缩放念头大小
- 向上滑动气泡 → 吹走/释放
- 向下拖动 → 放入念头罐暂存
- 摇一摇手机 → 所有气泡抖散（"shake off"）

---

#### 🧪 模块二：解钩实验室（Unhook Lab）— 引导式练习

**概述**：基于ACT/CBT/正念的结构化练习，以游戏化方式帮助用户学会认知解离。

**练习列表**（MVP 首批 6 个）：

| # | 练习名 | 时长 | 机制 |
|---|--------|------|------|
| 1 | 🚂 **念头列车** | 3min | 念头变成列车车厢，看它们驶过站台，你只是站台上的观察者 |
| 2 | 📺 **念头电视** | 2min | 念头变成电视屏幕上的字幕，你手持遥控器，可以调台、静音、关机 |
| 3 | ☁️ **云上书写** | 3min | 念头变成云朵上的文字，风吹过，文字渐渐散去 |
| 4 | 🎈 **气球释放** | 2min | 每个念头绑在一个气球上，松手让它飞走 |
| 5 | 🌊 **念头漂流瓶** | 3min | 念头装进瓶子，放入海浪，看它慢慢漂远 |
| 6 | 🔬 **念头显微镜** | 4min | 用"显微镜"观察念头的"微观结构"（拆解为词语→音节→符号→像素→消散）|

**每个练习的结构**：
```
引导语(文字+音频) → 输入念头 → 动画交互 → 呼吸锚定 → 结束反馈
```

---

#### 📊 模块三：觉察日志（Awareness Log）— 回顾与洞察

**概述**：自动记录用户的"解钩"历程，提供觉察维度的自我洞察。

**核心数据展示**：

**📅 时间线视图**：
- 每天捕捉了多少个念头
- 最常出现的念头主题（词云）
- 念头情绪分布（暖色/冷色比例）

**🔄 念头模式识别**：
- "你本周出现了 12 次关于「工作表现」的念头"
- "「我不够好」是你最常出现的核心念头"
- "周一的念头密度是周末的 3 倍"

**📈 解钩能力成长**：
- 从输入念头到释放的平均时间趋势
- "粘性"念头（反复出现的）追踪
- 每周"解钩指数"变化曲线

**💡 温柔洞察**（非诊断性）：
- "你和这个念头相处的时间越来越短了 ☀️"
- "你正在变得更擅长「看见」而非「被卷入」"

---

#### 🌙 模块四：睡前模式（Night Mode）— 特殊场景

**概述**：专门为睡前"大脑清空"设计的简化体验。

**流程**：
```
1. 深色界面 + 极简交互
2. 提示："把今天还在脑子里转的东西都倒出来"
3. 快速连续输入念头（语音/文字）
4. 所有念头变成萤火虫般的光点
5. 配合缓慢呼吸引导
6. 光点逐个熄灭（念头逐个告别）
7. 屏幕渐暗 → "晚安，你已经放下了今天"
8. 可选：播放白噪音/自然音助眠
```

## 四、技术架构方案

### 4.1 技术选型

| 层次 | 技术方案 | 理由 |
|------|---------|------|
| **前端框架** | React + TypeScript | 生态成熟，组件化开发 |
| **动画引擎** | Framer Motion + Three.js (可选) | 气泡物理效果，沉浸感 |
| **状态管理** | Zustand | 轻量，适合此类交互应用 |
| **本地存储** | IndexedDB (Dexie.js) | 念头数据本地优先，隐私安全 |
| **音频处理** | Web Audio API | 变声效果、白噪音生成 |
| **语音识别** | Web Speech API / Whisper API | 语音输入转文字 |
| **AI 辅助** | OpenAI API (可选) | 念头命名、模式识别 |
| **部署** | PWA | 可安装到手机桌面，离线可用 |


### 4.3 核心数据模型

```typescript
// 念头（Thought）
interface Thought {
  id: string;
  content: string;                    // 念头文字内容
  createdAt: Date;                    // 产生时间
  emotion: EmotionType;               // 情绪类型
  intensity: number;                  // 强度 1-10
  stickiness: number;                 // 粘性 1-10（越高越难放下）
  tags: string[];                     // 标签（自动+手动）
  status: 'active' | 'stored' | 'released';  // 状态
  releasedAt?: Date;                  // 释放时间
  releaseMethod?: ReleaseMethod;      // 释放方式
  recurrenceCount: number;            // 重复出现次数
}

// 情绪类型
type EmotionType = 
  | 'anxiety'     // 焦虑
  | 'anger'       // 愤怒
  | 'sadness'     // 悲伤
  | 'fear'        // 恐惧
  | 'guilt'       // 内疚
  | 'shame'       // 羞耻
  | 'neutral'     // 中性
  | 'mixed';      // 混合

// 释放方式
type ReleaseMethod = 
  | 'blow'        // 吹走
  | 'melt'        // 融化
  | 'float'       // 飘走
  | 'shrink'      // 缩小消失
  | 'transform'   // 变形
  | 'store';      // 暂存

// 解钩会话（Session）
interface UnhookSession {
  id: string;
  startedAt: Date;
  endedAt: Date;
  thoughts: Thought[];
  practiceType?: PracticeType;        // 使用的练习
  moodBefore: number;                 // 会话前心情 1-10
  moodAfter: number;                  // 会话后心情 1-10
  duration: number;                   // 持续秒数
}

// 练习类型
type PracticeType = 
  | 'thought-train'
  | 'thought-tv'
  | 'cloud-writing'
  | 'balloon-release'
  | 'drift-bottle'
  | 'microscope';
```

## 五、UI/UX 设计方向

### 5.1 设计原则

| 原则 | 描述 |
|------|------|
| **极简沉浸** | 界面尽可能少元素，让"空间感"成为主角 |
| **温暖安全** | 色调柔和，交互无压力，没有"打卡"压力 |
| **物理触感** | 动画有真实的物理反馈（弹性、重力、摩擦） |
| **无评判** | 不评判念头的好坏，只帮助"看见" |
| **渐进引导** | 首次使用温和引导，不堆砌功能 |

### 5.2 视觉风格

- **整体风格**：暗色系 + 柔光 + 毛玻璃
- **主色调**：深蓝/深紫（安静的夜空感）
- **点缀色**：气泡使用情绪对应色彩（半透明渐变）
- **字体**：中文使用思源黑体/霞鹜文楷，英文使用 Inter
- **图标**：线性图标 + 微动效
- **背景**：可选主题（星空/海底/森林/虚空白）

## 八、差异化与竞品分析

### 8.1 与现有产品的差异

| 维度 | 冥想App（Calm等）| 情绪日记App | **念头解钩器** |
|------|----------------|------------|-------------|
| 核心机制 | 被动听引导 | 记录情绪 | **主动与念头互动** |
| 用户角色 | 听众 | 记录者 | **观察者+操作者** |
| 念头处理 | 忽略/接纳 | 分析/归因 | **物化+解钩+释放** |
| 互动性 | 低 | 低 | **高（物理交互）** |
| 即时反馈 | 弱 | 弱 | **强（视觉+触觉）** |
| 使用门槛 | 需20min+ | 需文字表达 | **30秒即可完成一次** |

### 8.2 核心竞争力

1. **具身化**：不只是"想"，而是"看见、触碰、操作"
2. **即时性**：30秒一个念头，不需要长时间投入
3. **非诊断性**：不贴标签、不分析原因，只帮你"看见"
4. **美感**：不是医疗工具感，而是艺术化的体验
5. **科学根基**：基于ACT认知解离技术，有循证支持

### 9.2 传播策略

- **分享功能**：生成"今日解钩报告"卡片（精美图片，可分享到社交媒体）
- **念头艺术**：将释放的念头生成抽象艺术画（可保存/分享）
- **社区功能**（远期）：匿名念头广场，看看别人也有类似的念头

## 十、隐私与伦理

### 10.1 隐私原则
- 🔒 **本地优先**：所有念头数据默认存储在本地设备
- 🚫 **不上传**：不将用户念头内容上传到云端（除非用户主动开启同步）
- 🗑️ **可删除**：用户随时可以删除所有数据
- 👁️ **透明**：明确告知数据存储位置和使用方式

### 10.2 伦理边界
- ⚠️ App 明确声明「不是心理治疗工具」
- 🆘 检测到高风险内容（自残/自杀相关）时，温和引导至专业求助资源
- 💚 不制造焦虑，不设打卡机制，不搞连续使用天数
