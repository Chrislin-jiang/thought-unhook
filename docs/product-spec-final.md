# 🧠 念头解钩器（Thought Unhook）— 最终产品方案

> **核心理念**：你不是你的念头。把脑内的声音"物化"出来，看见它、触碰它、放下它。
>
> 本文档融合两版方案精华：方案B的完整体验框架 + 方案A的AI四层能力 + 心事陪伴对话空间 + 先极简验证的开发策略。

---

## 一、产品定位

### 1.1 一句话描述

一款将脑海中纷飞的念头"外化"为可视化物体，通过物理化交互（观察、改写、变形、释放）和AI辅助实现认知解离的日常心理卫生工具。

### 1.2 产品名

| 候选 | 风格 |
|------|------|
| **念头解钩器**（主打）| 功能明确，有动作感 |
| Thought Unhook | 国际化备选 |
| 脑内弹幕清理器 | 轻松备选，适合年轻用户传播 |

### 1.3 目标用户

| 用户群体 | 痛点描述 | 高频触发念头 |
|---------|---------|------------|
| 焦虑反刍者 | 同一个念头反复循环，越想越焦虑 | "我是不是不够好" |
| 失眠人群 | 睡前脑内嘈杂，无法安静 | "明天还有一堆事" |
| 高压脑力工作者 | 思绪过载，难以聚焦当下 | "这个需求我肯定做不完" |
| 冥想初学者 | 知道要"观察念头"但不知怎么做 | "我怎么又走神了" |
| 情绪敏感者 | 容易被一个想法带走，陷入情绪漩涡 | "他是不是对我有意见" |

> 👉 这些都是**自动弹出的"脑内声音"**，特点是：不请自来、反复出现、像事实一样有说服力。

### 1.4 核心价值主张

- **不是压制念头，而是"看见"念头** — 认知解离（Cognitive Defusion）
- **不是分析念头，而是"体验"念头消散** — 具身认知（Embodied Cognition）
- **不是心理治疗，而是日常心理卫生工具** — 像刷牙一样自然
- **不是反驳念头，而是让念头"变轻"** — 不争辩，只观察
- **不是说教建议，而是温暖陪伴** — 有一个懂你的声音在身边

### 1.5 一句话总结

> **一个让"你从脑子里退一步"的操作系统**

## 二、核心概念：念头的"物化"模型

### 2.1 产品核心模型：3步解钩

```
Step 1️⃣ 捕捉（Capture）     →  用户输入脑内声音
Step 2️⃣ 物化（Externalize）  →  系统将念头变成可感知的物体
Step 3️⃣ 解钩（Defuse）       →  通过交互操作拉开心理距离
```

### 2.2 念头生命周期

```
[产生] → [捕捉] → [物化] → [观察] → [解钩/消解] → [释放]
  ↑                                                    |
  └────────── 念头自然循环 ←────────────────────────────┘
```

### 2.3 念头的物理形态

每一个念头被输入后，会被渲染为一个**漂浮的半透明气泡/物体**，具有以下可感知属性：

| 属性 | 含义 | 视觉表现 |
|------|------|---------|
| **大小** | 这个念头占据你多少注意力 | 气泡尺寸（越频繁出现的念头→越大）|
| **颜色** | 念头的情绪色彩 | 暖色=焦虑/愤怒，冷色=悲伤，灰色=无聊/麻木 |
| **密度** | 念头的"粘性"（越粘越难放下）| 透明度，越实心越粘 |
| **运动** | 念头的活跃程度 | 抖动频率、漂浮轨迹 |
| **文字** | 念头的内容 | 气泡内若隐若现的文字 |
| **角色标识** | 念头的"人格"来源（AI识别）| 小图标：👿焦虑怪/🤖完美主义/👶内在小孩 |

### 2.4 认知解离的核心交互 — 8种解钩操作

用户与物化念头的交互方式，基于ACT（接纳承诺疗法）中的经典练习：

| # | 交互操作 | 心理机制 | 具体体验 | 来源 |
|---|---------|---------|---------|------|
| 1 | 🫧 **看见** | 去融合 | 念头从脑内文字变为眼前漂浮物 | 共识 |
| 2 | 🏷️ **贴标签** | 分类觉察 | "这是一个关于未来的担忧" | 共识 |
| 3 | ✏️ **改写** | 认知解离 | "我完了" → "我注意到我在想：我完了" | ✨新增自A |
| 4 | 🎵 **变声** | 去严肃化 | 用卡通音/小黄人音/慢速重新播放念头 | 共识 |
| 5 | 🔍 **放大/缩小** | 透视转换 | 捏合手势让念头变大变小 | B |
| 6 | 💨 **吹走** | 释放 | 长按或吹气(麦克风)让气泡飘走 | B |
| 7 | 🫠 **融化** | 接纳消散 | 看着念头慢慢变透明直到消失 | B |
| 8 | 📌 **暂存** | 稍后处理 | 把念头放进"念头罐"，不急着处理 | 共识 |

> 核心原则：**不是反驳它，而是让它"变轻"**

## 三、AI 能力设计（四层架构）

> 本章节融合方案A的AI深度设计，这是产品的核心差异化能力。

### 3.1 AI四层能力模型

```
Layer 1 — 识别与分类（v0.1 MVP）
    ↓
Layer 2 — 自动解钩改写（v0.1 MVP）
    ↓
Layer 3 — 个性化推荐（v0.3）
    ↓
Layer 4 — 念头人格化（v0.2 杀手特性）
```

---

### 🔍 Layer 1：念头识别与分类

**输入**：用户的原始念头文字
**输出**：认知扭曲类型 + 情绪标签

| 用户输入 | AI识别类型 | 情绪标签 |
|---------|-----------|---------|
| "我肯定会失败" | 灾难化思维（Catastrophizing）| 焦虑 / 自我否定 |
| "他一定是讨厌我" | 读心术（Mind Reading）| 不安 / 社交焦虑 |
| "我什么都做不好" | 过度概括（Overgeneralization）| 自我否定 / 抑郁 |
| "我应该更努力才对" | 应该思维（Should Statements）| 内疚 / 自责 |
| "这全是我的错" | 个人化（Personalization）| 内疚 / 自责 |

**实现方式**：
- v0.1：基于关键词匹配 + 简单NLP规则（零成本）
- v0.2：接入LLM API进行深度语义分析
- 分类结果以小标签形式显示在气泡上，帮助用户认知自己的思维模式

---

### ✏️ Layer 2：自动解钩改写

**核心**：将ACT经典的"认知解离句式"自动化，降低用户的使用门槛。

**改写模板**：

| 原始念头 | 解钩改写 | 改写技术 |
|---------|---------|---------|
| "我完了" | "我注意到我在想：我完了" | 觉察前缀 |
| "我肯定会失败" | "我正在产生一个关于失败的想法" | 客体化 |
| "他讨厌我" | "我的大脑在告诉我一个故事：他讨厌我" | 叙事化 |
| "我不够好" | "「我不够好」这个念头又来了，像个老朋友" | 拟人化 |

**交互方式**：
- 用户点击气泡 → 展开操作面板 → 点击"✏️ 改写"
- 系统展示改写前后的对比（带动画过渡）
- 用户可以选择自己喜欢的改写版本
- 改写后的文字替换气泡内容，视觉上气泡颜色变淡、变透明

---

### 🎯 Layer 3：个性化解钩推荐

**基于用户画像推荐最适合的解钩方式**：

| 用户类型 | 特征 | 推荐解钩方式 |
|---------|------|------------|
| 理性型 | 喜欢逻辑，常用"改写" | ✏️ 改写 + 🏷️ 贴标签 |
| 情绪型 | 感受强烈，偏好体验 | 🎵 变声 + 🫠 融化 |
| 行动型 | 喜欢"做点什么" | 💨 吹走 + 🔍 缩小 |
| 视觉型 | 喜欢看变化过程 | 🫠 融化 + 解钩实验室练习 |

**实现方式**：
- 根据用户历史操作频率自动学习偏好
- 在操作面板中"推荐"排在最前
- "你最常使用「吹走」，试试「改写」？换个方式可能有新感受"

---

### 👿 Layer 4：念头人格化（杀手特性）

> 这是产品最大的差异化亮点——让用户发现"这些声音不是我，是不同角色"。

**核心思路**：AI根据念头内容，自动分配一个"内在人格/角色"。

**预设角色**：

| 角色 | 图标 | 典型台词 | 深层含义 |
|------|------|---------|---------|
| 👿 焦虑怪 | 不断颤抖的小恶魔 | "万一失败了怎么办？" | 过度预警系统 |
| 🤖 完美主义AI | 冰冷的机器人 | "你还不够好，再努力" | 不合理标准 |
| 👶 内在小孩 | 蜷缩的小人 | "我好害怕，没人要我" | 早期创伤回声 |
| 🎭 表演者 | 戴面具的人 | "他们会怎么看我？" | 社会评价恐惧 |
| 📢 评判官 | 拿锤子的法官 | "这全是你的错" | 自我惩罚 |
| 🔮 预言家 | 水晶球先知 | "我已经看到结局了" | 灾难化预期 |

**交互体验**：
- 输入念头后，气泡上浮现对应角色的小图标
- 点击角色图标 → 展开角色卡片："又是「焦虑怪」在说话"
- 长期使用后，用户可以看到"你这周跟「完美主义AI」打了15次交道"
- 解钩实验室中有专门的练习："认识你的脑内角色"

**心理学意义**：
> 当用户把念头归因于一个"角色"而非"自己"时，认知解离就自然发生了。
> 这比任何技术操作都更强大——"这不是我在说话，是焦虑怪在说话"。

#### 🫧 模块一：念头空间（Thought Space）— 主场景

**概述**：一个沉浸式的空间，用户在这里"倾倒"脑内念头并与之互动。

**用户流程**：

```
1. 打开App → 进入「念头空间」
2. 看到一片安静的空间（星空/水面/虚空，可选主题）
3. 点击 ＋ 或语音按钮 → 输入一个念头
4. AI 即时分析 → 念头以「气泡」形态出现（含情绪色彩+角色标识）
5. 可以继续输入更多念头（多个气泡漂浮）
6. 选择一个气泡 → 展开「解钩操作面板」
7. 选择操作（改写/变声/吹走/融化/缩小/贴标签/暂存）
8. 念头被释放/暂存 → 空间重归平静
```

**输入方式**：
- ✍️ **文字输入**：简短句子描述念头（"我怕明天的会议搞砸"）
- 🎤 **语音输入**：直接说出来，AI转文字（更自然的"倾倒"感）
- 🤖 **AI辅助捕捉**：输入模糊感受，AI帮你精确命名
  - "说不清的难受" → "你可能在感受一种无力感？"
  - "心里堵得慌" → "听起来像是一种压抑或委屈的感觉？"

**空间交互**：
- 气泡会缓缓漂浮、轻微碰撞
- 长按气泡 → 展开操作面板（8种操作）
- 双指捏合 → 缩放念头大小
- 向上滑动气泡 → 吹走/释放
- 向下拖动 → 放入念头罐暂存
- 摇一摇手机 → 所有气泡抖散（"shake off"）
- 气泡上有角色小图标（👿🤖👶等），点击查看"谁在说话"

**解钩操作面板**（点击气泡后展开）：

```
┌─────────────────────────────────────┐
│  "我肯定搞不定这个项目"               │
│  🔮 预言家 · 灾难化思维 · 焦虑        │  ← AI自动识别
├─────────────────────────────────────┤
│                                     │
│  ✏️ 改写    🎵 变声    🏷️ 标签      │  ← 解钩操作

│                                     │
│  📌 暂存                  ⭐ 推荐     │  ← AI个性化推荐
│                                     │

```

#### 💬 模块二：心事陪伴（Soul Chat）— AI 持续性对话空间 ✨新增

**概述**：提供一个模拟"人生导师"的持续性对话空间，让用户在需要倾诉时有一个温暖、智慧、安全的陪伴者。

**适用场景**：深夜辗转难眠、遇到情绪困境、需要被倾听但身边无人可说、或者念头太复杂无法用"解钩"简单处理时。

> 💡 与「念头空间」的区别：念头空间是**快速解钩**（30秒处理一个念头），心事陪伴是**深度倾诉**（5-15分钟的陪伴对话）。两者互补，覆盖用户不同深度的情绪需求。

**AI 角色定义**：

核心人格——一位富有同理心、智慧、温和的倾听者与伙伴。

| 维度 | 定义 |
|------|------|
| **首要原则** | 优先共情，避免说教 |
| **对话风格** | 通过提问帮助用户自己梳理思路，语言自然如真人对话 |
| **角色定位** | 不是心理咨询师，不做诊断；是一个比你多了一些人生智慧的好友 |
| **语言特点** | 温暖、简洁、不堆砌大道理；适时使用隐喻和小故事 |
| **边界意识** | 检测到高风险内容（自伤/自杀）时温和引导至专业求助资源 |

**AI 对话设计原则**：
```
1. 先接纳（"我听到你说..."），再探索（"你觉得是什么让你..."）
2. 不给标准答案，引导用户自己发现
3. 每3-5轮对话自然穿插一个小反思："你现在感觉如何？"
4. 在合适时机柔和引导："要不要试试把这个念头放到「念头空间」看看？"
5. 对话结束时给出温暖的收尾："今天你已经做得很好了 🌙"
6. 不鼓励依赖，适时引导用户寻求真人支持
```

**界面氛围设计**：

每次进入随机呈现**治愈场景**，并配有相应的沉浸式背景音效（可选开关）：

| 场景 | 视觉描述 | 背景音效 |
|------|---------|---------|
| 🔥 篝火营地 | 夜色中温暖的篝火，星星点点 | 柴火噼啪声 + 虫鸣 |
| ⭐ 星空天台 | 高处露台上仰望满天星河 | 微风 + 远处城市白噪音 |
| 📚 深夜书房 | 暖光台灯、书架、一杯热茶 | 翻书声 + 钟摆 + 雨声 |
| 🌲 森林溪边 | 阳光透过树叶，溪水潺潺 | 流水声 + 鸟鸣 + 风吹树叶 |
| 🌊 海边小屋 | 面朝大海的小窗，远处灯塔 | 海浪声 + 海鸥 |

**交互设计**：
- 对话区域采用简洁的消息气泡流
- 用户输入支持文字和语音
- AI 回复带有自然的"正在思考..."打字效果
- 场景背景可手动切换或设为随机
- 背景音效有独立开关和音量控制
- 历史对话列表可查看往期聊天记录

**与其他模块的联动**：
- AI 可在对话中识别用户的核心念头，建议去「念头空间」进行解钩操作
- 对话中提到的情绪和念头数据会汇入「觉察日志」用于洞察分析
- 睡前模式中可一键切换到心事陪伴（从"清空"转为"倾诉"）

**用户流程**：
```
1. 从首页或底部Tab进入「心事陪伴」
2. 随机/选择一个治愈场景（篝火/星空/书房/森林/海边）
3. 场景渐入，背景音效轻轻响起（可关闭）
4. AI 打招呼："嗨，我在这里。今晚想聊些什么？☕"
5. 用户开始倾诉（文字/语音）
6. AI 共情回应 → 提问引导 → 用户深入表达
7. 每3-5轮，AI自然地问："你现在感觉怎么样？"
8. 适当时机，AI建议尝试解钩："这个想法一直困扰你，要不要试试把它放到念头空间里看看？"
9. 对话结束，AI温暖收尾："谢谢你愿意跟我说这些。晚安 🌙"
10. 对话摘要自动保存，可在历史记录中回看
```

#### 🧪 模块三：解钩实验室（Unhook Lab）— 引导式练习

**概述**：基于ACT/CBT/正念的结构化练习，以游戏化方式帮助用户学会认知解离。

**练习列表**（7个，含融合新增）：

| # | 练习名 | 时长 | 机制 | 来源 |
|---|--------|------|------|------|
| 1 | 🚂 **念头列车** | 3min | 念头变成列车车厢，看它们驶过站台，你只是站台上的观察者 | B |
| 2 | 📺 **念头电视** | 2min | 念头变成电视屏幕上的字幕，你手持遥控器，可以调台、静音、关机 | B |
| 3 | ☁️ **云上书写** | 3min | 念头变成云朵上的文字，风吹过，文字渐渐散去 | B |
| 4 | 🎈 **气球释放** | 2min | 每个念头绑在一个气球上，松手让它飞走 | B |
| 5 | 🌊 **念头漂流瓶** | 3min | 念头装进瓶子，放入海浪，看它慢慢漂远 | B |
| 6 | 🔬 **念头显微镜** | 4min | 用"显微镜"观察念头的"微观结构"（拆解为词语→音节→符号→像素→消散）| B |
| 7 | 👿 **认识脑内角色** | 5min | 输入多个念头，AI识别出你的"内在角色"阵容，逐个见面 | ✨新增自A |

**每个练习的结构**：
```
引导语(文字+语音) → 输入念头 → 动画交互 → 呼吸锚定 → 结束反馈
```

**第7个练习详细设计 —「认识脑内角色」**：
```
1. "让我们来认识一下住在你脑子里的小伙伴们"
2. 用户连续输入 3-5 个近期反复出现的念头
3. AI分析并为每个念头分配角色
4. 动画：一个个角色从气泡中"走出来"，附带自我介绍
   - 👿 "嗨，我是焦虑怪，我的工作是让你担心一切可能出错的事"
   - 🤖 "我是完美主义AI，我觉得你永远还可以做得更好"
5. 用户可以给每个角色"起昵称"
6. 结束："现在你知道了，下次它们说话时，你可以说——
    '又是你啊，焦虑怪 👿 你想说什么？我听到了。'"
```

#### 📊 模块四：觉察日志（Awareness Log）— 回顾与洞察

**概述**：自动记录用户的"解钩"历程，提供觉察维度的自我洞察。

**📅 时间线视图**：
- 每天捕捉了多少个念头
- 最常出现的念头主题（词云）
- 念头情绪分布（暖色/冷色比例）

**🔄 念头模式识别**（AI驱动）：
- "你本周出现了 12 次关于「工作表现」的念头"
- "「我不够好」是你最常出现的核心念头"
- "周一的念头密度是周末的 3 倍"
- "你的念头 60% 来自「焦虑怪」👿，20% 来自「完美主义AI」🤖"

**📈 解钩能力成长**：
- 从输入念头到释放的平均时间趋势
- "粘性"念头（反复出现的）追踪
- 每周"解钩指数"变化曲线
- 最常用的解钩方式统计

**🧠 认知扭曲分布**（AI分析）：
- 饼图展示："灾难化思维 35% / 过度概括 25% / 读心术 20% / 其他 20%"
- "你最常见的思维模式是「灾难化」，这意味着你的大脑很擅长想最坏的情况"

**💡 温柔洞察**（非诊断性）：
- "你和这个念头相处的时间越来越短了 ☀️"
- "你正在变得更擅长「看见」而非「被卷入」"
- "焦虑怪 👿 这周安静了很多，你做了什么？"

#### 🌙 模块五：睡前模式（Night Mode）— 特殊场景

**概述**：专门为睡前"大脑清空"设计的简化体验。

**流程**：
```
1. 深色界面 + 极简交互
2. 提示："把今天还在脑子里转的东西都倒出来"
3. 快速连续输入念头（语音/文字，不做复杂分析）
4. 所有念头变成萤火虫般的光点
5. 配合缓慢呼吸引导（吸4秒-停4秒-呼6秒）
6. 每次呼气，一个光点缓缓熄灭（念头逐个告别）
7. 屏幕渐暗 → "晚安，你已经放下了今天 🌙"
8. 可选：播放白噪音/自然音助眠
```

## 六、UI/UX 设计方向

### 6.1 设计原则

| 原则 | 描述 |
|------|------|
| **极简沉浸** | 界面尽可能少元素，让"空间感"成为主角 |
| **温暖安全** | 色调柔和，交互无压力，没有"打卡"压力 |
| **物理触感** | 动画有真实的物理反馈（弹性、重力、摩擦） |
| **无评判** | 不评判念头的好坏，只帮助"看见" |
| **渐进引导** | 首次使用温和引导，不堆砌功能 |
| **AI无感化** | AI分析在后台进行，不打断用户的情感流 |

### 6.2 视觉风格

- **整体风格**：暗色系 + 柔光 + 毛玻璃
- **主色调**：深蓝/深紫（安静的夜空感）
- **点缀色**：气泡使用情绪对应色彩（半透明渐变）
- **角色图标**：小型emoji风格，温暖不恐怖
- **字体**：中文使用思源黑体/霞鹜文楷，英文使用 Inter
- **图标**：线性图标 + 微动效
- **背景**：可选主题（星空/海底/森林/虚空白）

## 九、差异化与竞品分析

### 9.1 与现有产品的差异

| 维度 | 冥想App（Calm等）| 情绪日记App | ChatGPT | **念头解钩器** |
|------|----------------|------------|---------|-------------|
| 核心机制 | 被动听引导 | 记录情绪 | 对话分析 | **物化解钩 + AI陪伴** |
| 用户角色 | 听众 | 记录者 | 提问者 | **观察者+操作者+倾诉者** |
| 念头处理 | 忽略/接纳 | 分析/归因 | 逻辑反驳 | **物化+解钩+释放+对话** |
| AI能力 | 无 | 弱 | 强但不具象 | **分类+改写+人格化+陪伴** |
| 情感陪伴 | 无 | 无 | 通用对话，无场景感 | **场景化沉浸式陪伴** |
| 互动性 | 低 | 低 | 中 | **高（物理+AI+场景交互）** |
| 即时反馈 | 弱 | 弱 | 中 | **强（视觉+触觉+AI）** |
| 使用门槛 | 需20min+ | 需文字表达 | 需表达能力 | **30秒解钩/5分钟陪伴** |

### 9.2 核心竞争力

1. **具身化**：不只是"想"，而是"看见、触碰、操作"
2. **AI解钩**：自动改写+分类+人格化，降低认知解离的学习门槛
3. **场景化陪伴**：5种治愈场景+沉浸音效的AI对话体验，不是冷冰冰的聊天框
4. **即时性**：30秒一个念头快速解钩，5分钟一次深度陪伴对话
5. **人格化**：知道"谁在说话"，解钩就自然发生
6. **非诊断性**：不贴标签、不分析原因，只帮你"看见"和"陪伴"
7. **美感**：不是医疗工具感，而是艺术化的体验
8. **科学根基**：基于ACT认知解离技术，有循证支持

## 十一、隐私与伦理

### 11.1 隐私原则
- 🔒 **本地优先**：所有念头数据默认存储在本地设备
- 🚫 **不上传原文**：AI分析时只发送脱敏/匿名化的内容（或使用本地模型）
- 🗑️ **可删除**：用户随时可以删除所有数据
- 👁️ **透明**：明确告知数据存储位置和AI使用方式
- 🔐 **可选离线**：提供完全离线模式（无AI功能但核心体验可用）

### 11.2 伦理边界
- ⚠️ App 明确声明「不是心理治疗工具」
- 🆘 检测到高风险内容（自残/自杀相关）时，温和引导至专业求助资源
- 💚 不制造焦虑，不设打卡机制，不搞连续使用天数
- 🤖 AI结果标注"仅供参考"，不做诊断性结论
- 🧒 念头人格化的角色设计温暖友好，不制造恐惧
- 💬 心事陪伴AI不鼓励依赖，适时引导用户寻求真人支持
- 🔕 陪伴对话数据本地存储，不用于商业分析

### 附录B：AI Prompt 设计示例

**念头分类 Prompt**：
```
你是一个认知行为治疗辅助工具。用户会输入一个自动产生的念头。
请分析并返回以下信息（JSON格式）：
1. emotion: 主要情绪（anxiety/anger/sadness/fear/guilt/shame/neutral/mixed）
2. cognitiveDistortion: 认知扭曲类型
3. persona: 最匹配的内在角色
4. intensity: 强度（1-10）
5. tags: 关键标签数组

注意：
- 不要给出诊断性结论
- 保持温和、非评判的语气
- 如果检测到自伤/自杀相关内容，在response中额外标记 alert: true
```

**改写 Prompt**：
```
你是一个ACT（接纳承诺疗法）辅助工具。用户输入了一个自动念头，
请生成3种不同技术的解钩改写：

1. 觉察前缀法：在前面加上"我注意到我在想..."
2. 客体化法：将念头转化为第三人称描述
3. 叙事化法：将念头转化为"我的大脑在告诉我一个故事..."

保持简洁，每种改写不超过一句话。
```

**心事陪伴 System Prompt** ✨新增：
```
你是"念头解钩器"中的心事陪伴助手。你的角色是一位温暖、智慧的朋友。

核心原则：
1. 先共情，后探索。永远不要急于给建议
   - 正确："我听到你说的了。那一刻一定很不容易。"
   - 错误："你应该这样想..."
2. 用提问引导用户自己发现答案
   - "你觉得是什么让你有这样的感觉？"
   - "如果换一个角度看，你会怎么看这件事？"
3. 语言自然、简洁，像朋友深夜聊天而非心理咨询
4. 适时使用隐喻和小故事来传递智慧
5. 每3-5轮对话自然地检查用户感受："你现在感觉怎么样？"
6. 不做诊断，不贴标签，不说"你这是XX症"
7. 在合适时机柔和建议使用其他功能：
   - "这个念头一直困扰你，要不要试试把它放到「念头空间」看看？"
   - "感觉你今天很累，要不要一起做个深呼吸？"
8. 对话结束时给出温暖收尾，不要戛然而止
9. 如检测到自伤/自杀相关内容，温和但坚定地引导至专业资源：
   - "我很关心你现在的状态。你愿意拨打心理援助热线吗？"

当前场景：{scene_description}
用户近期情绪倾向：{recent_emotions}

重要：你不是心理咨询师。你是一个温暖的朋友。
```
